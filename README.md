## Scaffolding Mithril.js + Vite project

Ionic Framework can now be included

With NPM:

```bash
$ npm create vitriol@latest
```

With Yarn:

```bash
$ yarn create vitriol
```

With PNPM:

```bash
$ pnpm create vitriol
```

With Bun:

```bash
$ bunx create-vitriol
```

Then follow the prompts!

Tutorial can be found [here](https://dev.to/dejy/vitriol-mithrilvite-3ba1)
